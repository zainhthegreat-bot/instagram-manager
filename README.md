# Instagram Manager

A comprehensive tool for managing Instagram accounts, posts, direct messages, and comments using the Meta Business API.

## Features

- **Post Management**: Create, schedule, and manage Instagram posts
- **DM Handling**: Read and respond to Instagram Direct Messages
- **Comment Management**: Monitor and respond to comments across posts
- **Analytics**: Track engagement and performance metrics
- **Account Oversight**: Manage multiple Instagram accounts from one interface

## Prerequisites

- Meta Developer Account
- Instagram Business or Creator Account
- Valid API credentials for Meta Business API

## Installation

1. Clone this repository
2. Configure your Meta API credentials
3. Run the application
